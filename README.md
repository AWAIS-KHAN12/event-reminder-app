# Smart Event Reminder Application

A cross-platform mobile application developed using Flutter and Dart that helps users efficiently manage schedules, reminders, and tasks using Artificial Intelligence.

## Project Overview

The Smart Event Reminder Application simplifies event scheduling through Natural Language Processing (NLP). Instead of manually selecting dates and times, users can simply type instructions in plain English.

Example:
"Meeting with Sir Ali next Tuesday at 10 AM"

The application processes the input using the Gemini API and automatically extracts:
- Event Title
- Date
- Time

The reminder is then scheduled automatically.
